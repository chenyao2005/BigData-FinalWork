# 🗽 纽约宜居方程：数据驱动的街区选址引擎

> **在线体验：** [https://chenyao2005.github.io/BigData-FinalWork/](https://chenyao2005.github.io/BigData-FinalWork/)

本项目是一次基于纽约市微观数据的交互式长文叙事实践。我们将城市切片，从社会人群、经济成本、自然生态与时空脉络四个维度，对纽约的邮政编码区域（ZIP Code）进行了深度的数据体检，旨在帮助不同画像的都市人群寻找最适合自己的“专属宜居地”。

本仓库包含该长文主站及其内嵌的所有可视化交互子应用的**完整静态部署代码**。

---

## 📂 目录结构说明

本仓库为可独立部署的静态站点：根目录即站点入口，各篇章图表以模块化子应用的形式存放在 `apps/` 目录下。

| 核心路径 | 说明 |
|------|------|
| `index.html` <br> `main.js` <br> `styles.css` | **主站入口与核心逻辑**。采用原生 HTML/JS/CSS 编写，通过动态解析 JSON 加载叙事框架并嵌入图表 iframe。 |
| `documentation.html` | **项目说明文档**。包含总问题陈述、设计方案、数据来源及团队分工。 |
| `data/site-content.json` | **站点总配置台**。控制各章节的标题、导语、交互指南、段落文案以及 iframe 的挂载路径。 |
| `output/` | **全局汇总数据**。目前存放第四篇章（微观生态）依赖的聚合数据。 |

### `apps/` 嵌入式子应用索引

子应用按照叙事顺序排列，均为经过打包或独立可运行的静态前端模块：

| 编号 | 对应篇章 | 模块名称 | 技术栈 / 说明 |
|------|----------|----------|---------------|
| `01-population-yearly` | 第一篇章：社会底色 | 人口年度变化 | D3.js 时序图与网格 |
| `02-population-rings` | 第一篇章：社会底色 | 人口圈层图 | D3.js 空间与比例联动 |
| `03-education-level` | 第一篇章：社会底色 | 教育水平 | Leaflet 分层地图 |
| `04-safety-linebox` | 第一篇章：社会底色 | 公共安全 | D3.js 折线与箱线图 |
| `05-economy` | 第二篇章：经济账本 | 经济与房租 | Vue + 散点/地图双向联动 |
| `06-transport` | 第三篇章：时空脉搏 | 交通与等时圈 | Vue + D3 多模态网络分析 |
| `07-environment` | 第四篇章：微观生态 | 环境雷达与地图 | 原生 JS + 综合打分管线 |
| `08-livability-ranking`| 终章：专属宜居地 | 综合选址系统 | Vue 3 动态堆叠柱状图 |

*(更详细的构建历史与原始项目对应关系，可参见 `docs/目录结构说明.md`。)*
